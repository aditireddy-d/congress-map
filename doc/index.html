Constants Reference (data.js)

CAPITALS

State capital coordinates [lat, lng] for all 50 states. Used as fallback Senate dot placement.

Type: Object.<string, Array<number>> — 50 entries

DISTRICTS

Congressional district geographic centroids [lat, lng] for all 435 House districts. Keys in "STATE-DISTRICT" format (e.g., "CA-12").

Type: Object.<string, Array<number>> — 435 entries

ABBR_TO_NAME / NAME_TO_ABBR

Bidirectional state abbreviation ↔ full name mappings for all 50 states.

Type: Object.<string, string> — 50 entries each

STATIC_HOUSE / STATIC_SENATE

Static fallback representative counts used when the API is unavailable. STATIC_HOUSE reflects 119th Congress apportionment. STATIC_SENATE is always 2 per state.

Type: Object.<string, number>
